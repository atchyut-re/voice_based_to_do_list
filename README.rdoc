== Voice Based to do list generator built in rails using HTML, CSS, Javascript

* Clone the app
* Run in the sever
* Try saying command "I have to <-task to do->"

Eg: I have to "Book tickets for tomorrow trip"

Note: This app don't save any of the info in DB. So, make sure you save your info before you refresh or close the tab.
